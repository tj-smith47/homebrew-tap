# Homebrew Tap

Homebrew formulae for [cfgd](https://github.com/tj-smith47/cfgd) and [shelly-cli](https://github.com/tj-smith47/shelly-cli).

## Installation

```bash
brew tap tj-smith47/tap
brew install cfgd
brew install shelly-cli
```

Or install directly:

```bash
brew install tj-smith47/tap/cfgd
brew install tj-smith47/tap/shelly-cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `cfgd` | Encrypted, git-native configuration management |
| `shelly` | Command-line interface for Shelly smart home devices |

## Updating

```bash
brew update
brew upgrade cfgd
brew upgrade shelly-cli
```
