# Homebrew Tap

Homebrew formulae for [shelly-cli](https://github.com/tj-smith47/shelly-cli) and related tools.

## Installation

```bash
brew tap tj-smith47/tap
brew install shelly-cli
```

Or install directly:

```bash
brew install tj-smith47/tap/shelly-cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `shelly` | Command-line interface for Shelly smart home devices |

## Updating

```bash
brew update
brew upgrade shelly-cli
```

## Issues

If you encounter problems with the formulae, please open an issue on this repository.

For issues with the CLI itself, please report them at [shelly-cli issues](https://github.com/tj-smith47/shelly-cli/issues).
